SimpleSetup

Automated system configuration for Linux, Windows, and macOS. One command, zero stress.

bash <(curl -s https://setup.carino.systems/setup.sh)

Supports Debian, Ubuntu, Fedora, openSUSE, RHEL & CentOS.

iwr -useb https://carino.systems/setup.ps1 | iex

Optimized for Windows 10 and 11 via PowerShell.

bash <(curl -s https://setup.carino.systems/setup.sh)

Testing support for Homebrew and system defaults. Use with caution.

Smart Detection

Automatically identifies your OS, distribution, and hardware to install the right components.

Fully Localized

Menus available in English, Japanese, Russian, Spanish, Finnish, Chinese, Korean, and Hebrew.

Purpose-Built

Choose between Quick, Server, or specialized setups for Astronomy, Design, and more.

Available Arguments

Flag Action
quickInstalls Technical Setup and Basic Purpose packages.
serverMinimal server setup with essential dev tools.
nvidia/amd/intelForces specific graphic driver installation.
desktopOpens a menu to select a new Desktop Environment.
